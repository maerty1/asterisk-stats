# Адаптеры базы данных

Эта директория содержит адаптеры для работы с базой данных.

## Поддерживаемый адаптер

### mysql2 (используется по умолчанию)

- **Пакет**: `mysql2`
- **Преимущества**: Максимальная производительность, нативный драйвер MySQL
- **Использование**: Приложение использует этот адаптер по умолчанию

Все адаптеры реализуют интерфейс `BaseAdapter` и обеспечивают единый API для работы с базой данных.

## Структура

- `base-adapter.js` - Базовый класс для всех адаптеров
- `mysql2-adapter.js` - Реализация адаптера для mysql2
