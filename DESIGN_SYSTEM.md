# Система дизайна Asterisk Queue Analytics

## Обзор
Приложение использует единую систему дизайна на основе Material Design 3 с адаптацией для колл-центра.

## Цветовая палитра

### Основные цвета
- **Primary**: #0061a6 (синий)
- **Secondary**: #535f70 (серый)
- **Accent**: #00acc1 (бирюзовый)
- **Success**: #4caf50 (зеленый)
- **Warning**: #ff9800 (оранжевый)
- **Error**: #f44336 (красный)
- **Info**: #2196f3 (голубой)

### Поверхности
- **Background**: #fdfbff (белый с легким оттенком)
- **Surface**: #fdfbff
- **Surface Variant**: #dbe1e9 (светло-серый)

## Типографика

### Шрифты
- Основной: Roboto
- Моноширинный: Monaco, Menlo, Ubuntu Mono

### Размеры
- h1: 1.5rem
- h2: 1.25rem
- h3: 1.125rem
- h4: 1rem
- h5: 0.875rem
- h6: 0.75rem
- Body: 0.875rem

## Отступы (Spacing)

- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- xxl: 48px

## Радиусы скругления

- sm: 4px
- md: 8px
- lg: 12px
- xl: 16px
- full: 50px

## Компоненты

### Кнопки
Все кнопки используют единый стиль с:
- Плавными переходами (0.2s)
- Hover эффектами
- Focus состояниями
- Disabled состояниями

### Формы
Все поля ввода имеют:
- Единый стиль границ
- Focus состояния с подсветкой
- Disabled состояния
- Валидацию

### Карточки статистики
- Единый стиль с тенями
- Цветовая индикация (success/warning/danger)
- Прогресс-бары
- Hover эффекты

## Анимации

- Fade In Up: для появления секций
- Slide In: для alert сообщений
- Draw Trend: для линии тренда
- Плавные переходы: 0.2s для всех интерактивных элементов

## Адаптивность

- Desktop: > 1024px
- Tablet: 768px - 1024px
- Mobile: < 768px

Все компоненты адаптивны и используют flexbox/grid.
